Comprobar si tu Web cumple con responsive design

El diseño web adaptativo o adaptable (en inglés, Responsive Web Design) es una técnica de diseño y desarrollo web que mediante el uso de estructuras e imágenes fluidas, así como de media-queries en la hoja de estilo CSS, consigue adaptar el sitio web al entorno del usuario.

Hoy en día existen diversos dispositivos para conectarse a internet: móviles, ipods, ipads, portátiles, etc., y cada dispositivo tiene una resolución de pantalla distinta lo que impide que una página Web que no use las técnicas de diseño web adptativo no se vea correctamente o parte de ella no quede visible a simple vista.

En este artículo queremos dar mención a unas herramientas online para revisar si tu Web cumple con el propósito del diseño web adaptativo:
